Zyrln VPS install package
==========================

Full guide: https://github.com/ajavadinezhad/zyrln/blob/main/README.md
(VPS section: Step 2 — Option B)

Quick start (run in this folder after unzip):

  ./install-vps-relay.sh user@YOUR_VPS_IP

Set ZYRLN_RELAY_KEY when installing — same value as EXIT_RELAY_KEY in Apps Script.

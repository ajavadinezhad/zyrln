# شروع سریع با Zyrln

Zyrln یک ابزار قدرتمند برای عبور از سانسور اینترنتی مبتنی بر **DPI** (Deep Packet Inspection) است. این ابزار با استفاده از تکنیک **Domain-fronting**، ترافیک شما را از طریق زیرساخت گوگل هدایت می‌کند تا به اینترنت باز و بدون محدودیت دسترسی داشته باشید.

در این راهنما، قدم‌های اولیه برای نصب و راه‌اندازی Zyrln را یاد می‌گیرید.

## پیش‌نیازها

قبل از شروع، به موارد زیر نیاز دارید:

1.  **یک حساب گوگل:** برای راه‌اندازی بخش Apps Script (رله ورودی).
2.  **یک VPS (سرور مجازی) یا یک حساب Cloudflare:** با یک IP عمومی برای اجرای رله خروجی. (در صورت تمایل می‌توانید از Cloudflare Worker به عنوان جایگزین رایگان استفاده کنید).
3.  **یک کلید احراز هویت (Auth Key):** این کلید، امنیت ارتباط بین تمام اجزای Zyrln را تضمین می‌کند.

## مرحله 1: تولید کلید احراز هویت

یک کلید منحصر به فرد تولید کنید. این کلید را در تمام مراحل بعدی (Apps Script، VPS و کلاینت) استفاده خواهید کرد.

در ترمینال (کامند لاین) دستگاه خود، دستور زیر را اجرا کنید:

```bash
openssl rand -base64 32
```

خروجی چیزی شبیه به این خواهد بود:
```text
swrkwbMS1X666fjzReip+PbodKcPyDK7Xbk5gRSgRUE=
```
::: warning مهم
کلید تولید شده را برای استفاده در مکانی امن نگهداری کنید
:::

## مرحله 2: راه‌اندازی رله Apps Script (ورودی گوگل)

### قدم اول: راه اندازی رله
1. به script.google.com بروید و یک پروژه جدید (New project) بسازید.
2. تمام محتوای فایل relay/apps-script/Code.gs را از مخزن Zyrln کپی کرده و در پروژه جدید خود جایگذاری کنید.
3. مقادیر ثابت (constants) بالای فایل را به صورت زیر تنظیم کنید:

```code.gs
const AUTH_KEY = "CHANGE_ME_TO_A_LONG_RANDOM_SECRET";
const EXIT_RELAY_URL = "https://CHANGE_ME_EXIT_RELAY_URL/relay";
const EXIT_RELAY_KEY = "";
```

**CHANGE_ME_TO_A_LONG_RANDOM_SECRET**: کلید تولید شده در مرحله 1 را به جای این عبارت قرار دهید.

**EXIT_RELAY_URL**: این عبارت پس از انتخاب متد رله مقدار دهی میشود.


### قدم دوم: انتخاب خروجی رله
بسته به نیاز خود میتوانید از دو روش زیر نحوه ارتباط خود را با دنیای آزاد اینتر مشخص کنید. لطفا با توجه به جدول زیر یکی از روش ها زیر را انتخاب کنید:

1. [🚀 راه‌اندازی VPS](/setup/vps)
2. [☁️ راه‌اندازی Cloudflare](/setup/cloudflare)

| ویژگی | 🖥️ VPS | ☁️ Cloudflare (cfw) |
|-------|--------|---------------------|
| **IP ثابت** | ✅ | ❌ |
| **ChatGPT و سایت‌های کلودفلری** | ✅ | ❌ |
| **تمام سایت‌ها (غیر کلودفلری)** | ✅ | ✅ |
| **رد شدن کپچا** | ✅ | ❌ |

### قدم سوم: ادامه راه اندازی رله

4. روی دکمه Deploy (بالا سمت راست) کلیک کنید.
5. گزینه New deployment را انتخاب کنید.
6. در پنجره باز شده، نوع deployment را Web app انتخاب کنید.
7. تنظیمات را به این صورت انجام دهید:
    * Descripton: My App
    * Execute as: Me
    * Who has access: Anyone
8. روی Deploy کلیک کنید.
9. آدرس (URL) ایجاد شده را کپی کنید.
    * چیزی شبیه این خواهد بود:
    * `https://script.google.com/macros/s/AKfycb.../exec`

::: warning مهم
آدرس را نگهداری کنید در مراحل بعد از آن استفاده میکنیم.
:::
